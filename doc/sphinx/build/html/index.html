Parameters

  • layers (int list) – Number of neuron per layer. The first number describes the input layer and the last number describes the output layer

  • lr (double) – Learning Rate for backpropagation

  • weights_list – List of the different weight matrixes. The matrix of index i is used to go from layer i to layer i + 1

  • pipbias_list – List of the different bias vector. The vector of index i is used to go from layer i to layer i + 1

  • weigths_and_bias (numpy matrixes list list) – Used to create a Neural Network instance with a specific set of weight
